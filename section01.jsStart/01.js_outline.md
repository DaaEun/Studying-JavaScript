# 01) 01. 자바스크립트 개요

#### JavaScript는 객체(object) 기반의 스크립트 언어이다.
		
- HTML로는 웹의 내용을 작성하고, CSS로는 웹을 디자인하며, JavaScript는 웹의 동작을 구현할 수 있다.

- 자바스크립트는 주로 웹 브라우저에서 사용되나, Node.js와 같은 프레임워크를 사용하면 서버 측 프로그래밍에서도 사용할 수 있다.
