# 07) 27. 객체의 개념

### 자바스크립트 객체

**객체** 란 이름(name)과 값(value)으로 구성된 프로퍼티(property)의 정렬되지 않은 집합이다.

프로퍼티의 값으로 함수가 올 수도 있는데, 이러한 프로퍼티를 **메소드(method)** 라고 한다.
