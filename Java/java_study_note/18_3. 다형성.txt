* 다형성(Polymorphism)


- 다형성이란 "객체가 여러 형태를 가진다"라는 의미로 해석되며, 하나의 객체가 
여러가지 유형으로 사용되는 것을 의미합니다.


- 다형성은 상속을 전제조건으로 합니다.


- 다형성을 위해 자바는 자식클래스가 부모클래스의 타입을 가질 수 있도록 허용합니다. 즉, 부모 타입에 모든 자식객체가 대입될 수 있습니다.







* 이종모음(Heterogeneous Collection)

- 이종모음은 배열에 다형성을 적용시키는 원리입니다. 배열은 원래 동종모음 구조입니다.

- 예를 들어 int[] iArr = new int[10]; 이런 구조의 배열이 있다면
iArr배열에는 int형 정수 데이터만 저장할 수 있습니다.

- 하지만 다형성을 이용하면 이종모음 구조의 객체 배열이 생성 가능합니다.







* 매개 변수의 다형성

- promotion은 멤버 변수의 값을 대입할 때도 발생하지만, 메서드를 호출할 때 
사용하는 매개 변수에도 발생할 수 있습니다.


- 보통 메서드를 호출할 때는 메서드 선언부에서 지정한 데이터 타입과 일치하는 
매개값을 전달하여 호출하지만, 매개 변수에 다형성을 적용하면 자식 객체를 전달할 수도 있습니다.
