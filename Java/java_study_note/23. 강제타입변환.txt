* 강제 타입 변환(Type Casting)


- 강제 타입 변환은 부모 타입을 자식 타입으로 변환하는 것을 말합니다.


- 객체에서 타입 캐스팅을 사용하려면 우선 먼저 Promotion이 일어나야 합니다. 
즉, 부모 타입으로 한번 형 변환이 된 자식 객체만 강제 타입 변환을 사용할 수 있습니다.


- Promotion이 일어나면 자식 클래스가 가지고 있는 재정의되지 않은 메서드를 
사용할 수 없다는 단점이 있었습니다.
-이 단점을 극복하기 위해 강제 타입 변환을 사용하여 자식 메서드를 호출하는 방법을 사용합니다.
